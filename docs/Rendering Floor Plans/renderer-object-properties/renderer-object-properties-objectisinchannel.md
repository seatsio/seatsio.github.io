**Param**: `channelKey`  
**Type**: `string`  
The key of the channel you want to check against.

**Returns**: `boolean`  
Whether the object is in the specified channel or not.

```javascript
object.isInChannel('8950dddc-e51f-c7cb-986a-2c5e728b8c21')
```
