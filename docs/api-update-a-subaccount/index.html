  • name: the new name of the subaccount. Optional. Can be any string (including spaces or special characters)
  • email: the new name of the subaccount. Optional. If you pass in a new e-mail address (which is different from the existing one), seats.io sends a mail to that address. That mail contains a link to a page which allows the user to choose a password to log in to the dashboard.
